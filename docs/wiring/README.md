# The Wiring Book

A complete reference for the Nuke wiring layer overlay system.
Read in order. Each chapter builds on the previous.

## Table of Contents

### Part I: Foundations
1. [The Workshop Model](chapters/01-workshop-model.md) — Vehicle, workspace, intelligence. Why the harness is derived, not designed.
2. [The Squarebody Factory Harness](chapters/02-factory-harness.md) — Every circuit in a 1973-1987 C/K truck. GM wire colors, fuse positions, known failures.
3. [The Tier System](chapters/03-tier-system.md) — From $1K Painless to $30K concentric-twist. Wire types, connector families, construction methods.
4. [The Supply Chain](chapters/04-supply-chain.md) — ProWire, Motec dealers, micro-suppliers. How parts move from catalog to workbench.

### Part II: The Build
5. [The Build Manifest](chapters/05-build-manifest.md) — Every device as a digital twin. Endpoints, prices, positions, suppliers.
6. [The Compute Engine](chapters/06-compute-engine.md) — How the system derives the harness. ECU selection, PDM assignment, wire sizing, color coding.
7. [The Motec Ecosystem](chapters/07-motec-ecosystem.md) — M130/M150/M1, PDM15/PDM30, LTCD, C125. Pin maps, firmware, CAN bus.
8. [The LS3 Sensor Suite](chapters/08-ls3-sensors.md) — Every sensor and actuator. ACDelco part numbers, connector types, pin counts.

### Part III: The Output
9. [The Cut List](chapters/09-cut-list.md) — Wire-by-wire bench document. Gauge, color, length, label, from→to.
10. [The Connector Schedule](chapters/10-connector-schedule.md) — Pin-by-pin for every connector. M150, PDM30, bulkheads.
11. [The Bill of Materials](chapters/11-bom.md) — Every part with catalog linkage, pricing, supplier, stock status.
12. [The Build Sheet](chapters/12-build-sheet.md) — Step-by-step assembly by harness section.

### Part IV: The Workspace
13. [The Professional Canvas](chapters/13-canvas.md) — Base layers, component markers, wire routing visualization.
14. [The Catalog Browser](chapters/14-catalog-browser.md) — Searching, placing, ordering parts.
15. [The Storefront](chapters/15-storefront.md) — Click to add, cart fills, order places, commission tracks.

### Appendices
