## 💻 WHAT'S HAPPENING BEHIND THE SCENES

### **Real-Time Database Updates**

```
Every order/trade triggers:

1. INSERT market_orders or UPDATE
2. MATCH against order book
3. INSERT market_trades (immutable)
4. UPDATE share_holdings (portfolio)
5. UPDATE vehicle_offerings (market data)
6. WebSocket BROADCAST (real-time UI)
7. PostgreSQL trigger → Supabase real-time
8. React subscriptions receive update
9. Component re-renders (< 100ms)
10. User sees live price change
```
