## 🎨 Design Tokens

### Colors
- **Background**: `#0a0a0a` (dark black)
- **Cards**: `#1a1a1a` (slightly lighter)
- **Borders**: `#222` (subtle)
- **Primary (Blue)**: `#4a9eff`
- **Success (Green)**: `#00c853`
- **Warning (Orange)**: `#ff9d00`
- **Error (Red)**: `#ff4444`
- **Text**: `#e0e0e0` (light gray)
- **Muted**: `#888` (medium gray)

### Typography
- **Heading**: 28px, 600 weight
- **Subheading**: 18px, 600 weight
- **Body**: 14-16px, 400 weight
- **Caption**: 12-13px, 400 weight

### Spacing
- **Section padding**: 40px
- **Card padding**: 20-30px
- **Grid gaps**: 4px (small), 8px (medium), 12px (large)
- **Button padding**: 12px 24px

### Effects
- **Border radius**: 4-12px
- **Border width**: 2px
- **Transitions**: 0.12s ease
- **Box shadow** (modals): `0 10px 40px rgba(0,0,0,0.5)`

## 📱 Mobile Responsive

### Adjustments for Mobile
- Single column grid
- Larger touch targets (48px minimum)
- Simplified toolbar (fewer buttons)
- Full-screen modals
- Swipe gestures for navigation
