## Platform-Specific Notes

### YouTube
- **API Access**: Full API available (YouTube Data API v3)
- **Rate Limits**: 10,000 units/day (free), 1M/day (paid)
- **Data Available**: Full metadata, transcripts (if enabled), comments
- **Best For**: Primary ingestion source

### TikTok
- **API Access**: Limited (Research API, requires approval)
- **Rate Limits**: Varies by tier
- **Data Available**: Basic metadata, engagement stats
- **Best For**: Secondary source, may need web scraping

### Instagram
- **API Access**: Graph API available
- **Rate Limits**: 200 calls/hour per user token
- **Data Available**: Posts, stories (limited), engagement stats
- **Best For**: Visual content, hashtag tracking

### Twitter/X
- **API Access**: API v2 available (paid tiers)
- **Rate Limits**: Varies by tier
- **Data Available**: Tweets, engagement, media
- **Best For**: Real-time updates, community engagement

## Ingestion Strategy by Creator Type

### Review Channels
- **Vehicle Detection**: High - titles/descriptions mention specific vehicles
- **Linking Confidence**: High - clear vehicle identification
- **Content Value**: High - detailed vehicle information

### Build Channels
- **Vehicle Detection**: Medium - project cars may not be fully identified
- **Linking Confidence**: Medium - need to track build progress
- **Content Value**: High - unique modifications, builds

### Collection Channels
- **Vehicle Detection**: High - specific vehicles in collection
- **Linking Confidence**: High - clear ownership/feature
- **Content Value**: Medium - showcase content, less technical

### Lifestyle Channels
- **Vehicle Detection**: Low - may feature multiple vehicles
- **Linking Confidence**: Low - vehicles may be borrowed/featured
- **Content Value**: Low - less vehicle-specific data
