# Page Schematics — Text-Based Wireframes

Every page gets a schematic BEFORE code is written.
A schematic is: what's on the page, why it's there, what happens when you click it.
No code. No colors. No CSS. Just structure and purpose.

Read the schematic. Agree on it. THEN build.
