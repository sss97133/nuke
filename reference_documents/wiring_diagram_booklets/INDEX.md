## 1987 Complete (ST-350) — Most comprehensive, 1988 model year

Uses grid-address wire coding system (alphanumeric A-Z rows, 1-37/75/113 columns).
Every wire labeled: circuit number, wire size (sq mm), wire color.

### Text Pages (pages 1-16, embedded in fold-out format)
Pages 1-2: TOC, wire coding description, tracing circuits example
Pages 3-5: Cruise control wiring example, headlamp troubleshooting
Pages 6-8: Basic electrical, circuit components, circuit diagnosis
Pages 9-10: Diagnostic tools, wire color abbreviations, wire size conversion
Pages 11-12: On-vehicle service, connector terminals, Weather-Pack
Pages 13-14: Weather-Pack/Metri-Pack procedures, wiring repair
Pages 15-16: Twisted wire repair, special tools (J-28742-A, J-22727, J-23520-B)

### Key Feature: 1988 Fuse Block Chart
25 fuses mapped with circuit numbers:
- Fuse 1: 20A, Cir 39 — Cruise, 4WD display, CVS timer, aux batt relay, seat belt buzzer, cluster ign feed
- Fuse 2: 15A, Cir 38/75 — Back-up lamps, turn signals
- Fuse 3: 10A, Cir 439 — T.C.C., cluster-speedo, gauges, air diverter, E.S.C., E.G.R., E.C.M. ign, RWAL brake switch
- Fuse 4: 10A, Cir 481 — Throttle body injectors
- Fuse 11: 25A, Cir 50 — H.V.A.C., rear wheel antilock, 4WD, aux battery relay
- Fuse 15: 15A, Cir 140 — Hazard flasher, seat belt buzzer, stop lamps, RWAL memory
- Fuse 18: Circuit not specified — specific to options
- Fuse 22: 10A, Cir 43 — Radio feed
- Fuse 25: 15A, Cir 806 — Crank, discreet
