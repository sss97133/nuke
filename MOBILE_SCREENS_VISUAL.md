## KEY MOBILE INTERACTION PATTERNS

### Swipe Gestures:
- **Left/Right:** Navigate images, timeline, tabs
- **Up:** Scroll content
- **Right from edge:** Go back
- **Pull down:** Refresh

### Tap Targets:
- **Minimum:** 44px × 44px
- **Buttons:** Full-width on mobile
- **Cards:** Tap anywhere to navigate

### Fixed Elements:
- **Top Header:** Always visible, 60px height
- **Bottom Nav:** Always visible, 60px height
- **Content Area:** Scrollable between fixed elements

### Loading States:
- **Skeleton screens** for data loading
- **Spinners** for actions
- **Progress bars** for uploads

### Error States:
- **Toast notifications** for transient messages
- **Inline errors** for form validation
- **Full-page errors** for critical issues
