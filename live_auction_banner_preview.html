How It Works

The Live Auction Banner automatically appears when a vehicle has an active auction listing. It shows real-time information including current bid, bid count, time remaining, and reserve price.

To see this banner on the live site: Create an active auction listing in the vehicle_listings table with status = 'active' and sale_type IN ('auction', 'live_auction').

INSERT INTO vehicle_listings ( vehicle_id, seller_id, sale_type, status, auction_end_time, current_high_bid_cents, bid_count, reserve_price_cents ) VALUES ( 'c1b04f00-7abf-4e1c-afd2-43fba17a6a1b', '[your-user-id]', 'auction', 'active', NOW() + INTERVAL '7 days', 2000000, -- $20,000 5, 2500000 -- $25,000 reserve );
